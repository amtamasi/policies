# policies

Privacy policies and support pages for my apps, served via GitHub Pages:

- https://amtamasi.github.io/policies/ — index + support
- https://amtamasi.github.io/policies/nextset/
- https://amtamasi.github.io/policies/bookmarked/
- https://amtamasi.github.io/policies/fitly/

Plain HTML, no build step. Each page is self-contained and must state only
what the app actually does — update the policy in the same change that
alters an app's data practices, before it ships.
